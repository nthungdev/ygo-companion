# ygo_companion

A mobile app to accompany Yu-Gi-Oh! players.

## Folder structure

```
lib/
  app.dart
  main.dart
  theme.dart
  providers/
  screens/
    screen_a/
    screen_b/
      widgets/
  widgets/
  models/
```

`lib/widgets/`
Common widget shared across the app

`lib/screens/<screen_name>/widgets/`
Widgets specific for <screen_name>
